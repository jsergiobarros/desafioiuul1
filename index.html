HTML5 - Hypertext Markup Language

Site Oficial

descrição

HTML (Hypertext Markup Language) é uma linguagem de marcação utilizada para criar e estruturar documentos na World Wide Web. Ele utiliza uma série de tags ou elementos para definir a estrutura e o conteúdo de uma página da web, como cabeçalhos, parágrafos, imagens, links e formulários. O HTML permite a criação de páginas da web estáticas e interativas, fornecendo uma base para a apresentação de informações e a interação do usuário na web.

Angular

Site Oficial

descrição

Angular é um framework de desenvolvimento front-end mantido pelo Google, projetado para a criação de aplicativos web dinâmicos e robustos. Utilizando a linguagem TypeScript, Angular oferece uma arquitetura baseada em componentes que facilita a construção de aplicativos modulares e reutilizáveis. Com recursos como roteamento, injeção de dependência, controle de formulários e comunicação com servidores via HTTP, Angular permite o desenvolvimento eficiente de aplicativos escaláveis e de alto desempenho. Além disso, o framework oferece uma variedade de ferramentas e bibliotecas adicionais para facilitar o desenvolvimento, melhorar a experiência do usuário e otimizar a renderização do lado do servidor. Em resumo, Angular é uma ferramenta poderosa para desenvolvedores que desejam criar aplicativos web modernos e sofisticados.

Vantagens

  • Arquitetura Baseada em Componentes
  • Produtividade com Angular CLI
  • Ecossistema Amplo e Maduro

limitações

  • Curva de Aprendizado
  • Desempenho em Aplicativos Pequenos
  • Tamanho do Pacote e Complexidade

CSS - Cascading Style Sheets

Site Oficial

descrição

CSS (Cascading Style Sheets) é uma linguagem de estilo que define a apresentação visual de documentos HTML e XML em uma página da web. Utilizando regras de estilo, o CSS permite controlar aspectos como cores, fontes, espaçamento, posicionamento e layout dos elementos HTML. Ele funciona aplicando essas regras aos elementos HTML por meio de seletores, separando a estrutura do conteúdo do seu estilo de apresentação. Isso promove uma maior flexibilidade, consistência e eficiência no design de páginas da web, permitindo aos desenvolvedores criar interfaces atraentes e responsivas para uma variedade de dispositivos e tamanhos de tela. Combinado com HTML e JavaScript, o CSS é uma ferramenta fundamental para a criação de experiências web modernas e visualmente atraentes.

Vantagens

  • Separação de Preocupações
  • PReutilização de Estilos
  • Reutilização de Estilos

limitações

  • Inconsistências entre Navegadores
  • Dificuldade em Controlar Layouts Complexos
  • Desempenho e Tamanho do Arquivo

JavaScript

Site Oficial

descrição

JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos, comumente utilizada para criar interatividade e dinamismo em páginas da web. Como uma das tecnologias essenciais da web, o JavaScript é executado no lado do cliente, ou seja, no navegador web do usuário, permitindo que os desenvolvedores criem experiências interativas, como validação de formulários, animações, manipulação do DOM (Document Object Model) e requisições assíncronas para atualizar partes específicas da página sem a necessidade de recarregar a página inteira. Além disso, JavaScript também é amplamente utilizado no desenvolvimento de aplicativos web, tanto no frontend quanto no backend, graças a frameworks e bibliotecas populares, como React, Angular, Vue.js e Node.js. Com sua flexibilidade e capacidade de se integrar a outras tecnologias, o JavaScript desempenha um papel crucial na construção de aplicativos web modernos e dinâmicos.

  • Interatividade e Dinamismo
  • Ecossistema Rico de Frameworks e Bibliotecas

React

Site Oficial

descrição

React é uma biblioteca JavaScript de código aberto mantida pelo Facebook, projetada para o desenvolvimento de interfaces de usuário (UI) interativas e reativas. Utilizando uma abordagem baseada em componentes, React permite aos desenvolvedores criar interfaces de usuário complexas dividindo-as em componentes independentes e reutilizáveis. Sua principal característica é o conceito de virtual DOM (Documento de Objeto Virtual), que permite atualizações eficientes da interface do usuário, minimizando a quantidade de manipulação direta do DOM real. Com React, os desenvolvedores podem criar aplicativos da web escaláveis e de alto desempenho, oferecendo uma experiência de usuário rápida e responsiva. Além disso, React é acompanhado de uma vasta comunidade de desenvolvedores e uma rica coleção de bibliotecas e ferramentas adicionais, como o React Router para roteamento e o Redux para gerenciamento de estado, tornando-o uma escolha popular para o desenvolvimento de aplicativos modernos.

Vantagens

  • Componentização
  • Virtual DOM
  • Ecossistema e Comunidade Ativa

limitações

  • Curva de Aprendizado Inicial
  • Complexidade de Configuração
  • Possível Sobrecarga de Abstração
